# Design Language

## Theme
Quiet operator dashboard.

## Visual principles
- dark base
- cool accent colors
- soft gradients only where useful
- rounded cards
- compact spacing with strong hierarchy
- badges/chips for source, fit, state, lists
- avoid noisy decorative motion

## Core UI objects
- fit badge
- role chips
- source chips
- lifecycle badges
- company trust badges
- black/white list markers
- chart panels
- explanation cards

## UX principles
- every page should answer “what should I do next?”
- important filters must be sticky and visible
- state changes should feel immediate
- navigation should not leave stale data visible
