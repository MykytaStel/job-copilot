# Vision

We are building a job search platform that helps users:

- discover jobs from multiple sources
- save and track applications
- understand job fit
- adapt CV/resume to vacancies
- receive better ranked results than simple keyword search

## MVP goals
- store and display jobs
- save jobs
- track application status
- basic search
- basic ranking
- one ingestion source
- stable web UI
- new Rust backend skeleton
- new Python ML service skeleton

## Not in MVP
- full custom LLM training
- advanced recruiter CRM
- full browser extension
- complex automation
